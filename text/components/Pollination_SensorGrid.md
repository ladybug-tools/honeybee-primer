Select SensorGrids from Pollination to use with Honeybee. Features: A - Convert Pollination SensorGrids into Honeybee Rooms; B - Internalise selected Pollination SensorGrids; C - Bake Pollination SensorGrids; D - Bake volatile Honeybee SensorGrids; E - Save Pollination SensorGrid identifiers if Internalised data is disabled.
